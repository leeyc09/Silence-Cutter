2-Pass ASR — Never Cuts Mid-Word

Pass 1 transcribes with Qwen3-ASR + ForcedAligner to get word-level timestamps. Pass 2 splits segments only at word end_time boundaries. The result: clean cuts with perfectly synced subtitles.

Pass 1: VAD → 30s chunks → ASR + Aligner → word timestamps
Pass 2: Split at word.end_time → never mid-word
